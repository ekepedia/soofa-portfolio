Formatting for Blog Posts

The blogging formatting follows standard HTML, but here are some essentials:


To make something bold, use the <b></b> tags

Hello, <b>this is bold</b>. This is not.

will show up as

Hello, this is bold. This is not.

To make something italicized, use the <i></i> tags

Hello, <i>this is italicized</i>. This is not.

will show up as

Hello, this is italicized. This is not.
